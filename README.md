
# Reporte Automático de Ventas - Python

Script Python que se conecta a SQL Server y genera automáticamente 
un reporte Excel con análisis de ventas.

## Tecnologías
- Python (pandas, pyodbc, openpyxl)
- SQL Server

## Contenido
- `reporte_ventas.py` — Script principal
- `reporte_ventas_20260521.xlsx` — Ejemplo de reporte generado

## Hojas del reporte
- Resumen Mensual — ingresos, costos y margen por mes
- Ranking Productos — productos ordenados por ganancia
- Ventas x Cliente — facturación total por cliente
